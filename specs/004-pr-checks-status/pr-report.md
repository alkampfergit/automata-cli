**Branch**: `004-pr-checks-status`

**Spec**: [specs/004-pr-checks-status/spec.md](spec.md)

## Summary

This feature extends the `automata git get-pr-info` command to display all CI/CD status checks associated with the pull request. Each check is shown with a pass/fail/pending symbol, and failed checks include their failure description inline. The `--json` output is extended with a `checks` array to support scripting and tooling.
