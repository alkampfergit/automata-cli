# PR Report: Get PR Open Comments

**Branch**: `006-get-pr-comments`
**Date**: 2026-03-31
**Spec**: [specs/006-get-pr-comments/spec.md](spec.md)

## Summary

Adds `automata git get-pr-comments`, a new subcommand that fetches and displays unresolved review thread comments for the current branch's GitHub pull request. Developers can now triage reviewer feedback directly from the terminal without switching to the GitHub web UI. The command supports both human-readable and `--json` output; Azure DevOps is not supported and the limitation is documented in `docs/azdo-gap.md`.
