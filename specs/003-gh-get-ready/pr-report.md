# PR Report: GitHub Issue Discovery and Auto-Implementation (get-ready)

**Branch**: `003-gh-get-ready`
**Date**: 2026-03-30
**Spec**: [specs/003-gh-get-ready/spec.md](./spec.md)

## Summary

This feature adds a `get-ready` command to automata-cli that discovers the next open GitHub issue matching a configured filter, posts a "working" comment to claim it, and invokes Claude Code locally to implement it. It also extends the configuration system with three new settings — issue discovery technique, filter value, and Claude system prompt — available only when GitHub mode is active.
